### Quick Start

1. **Clone the repository**
2. **Open the project**

**Method 1: Direct File Opening**

**Method 2: Using VS Code Live Server (Recommended)**
- Install "Live Server" extension in VS Code
- Right-click on `index.html`
- Select "Open with Live Server"
- Application opens at `http://localhost:5500`

**Advantages:**
- ‚úÖ Works without internet
- ‚úÖ No server costs
- ‚úÖ Instant data access
- ‚úÖ Data persists across sessions
- ‚úÖ No database setup required
- ‚úÖ Privacy - data stays on device

- ‚ö†Ô∏è Data stored per browser/device
